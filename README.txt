This zip file contains an example project with a destructible wall.

How to get here from the previous .zip
1. Add the Apex Destruction plugin through Edit -> Plugins -> Apex Destruction -> Enabled
2. Drag a cube to the content folder
3. Right click -> create destructable mesh
4. Fracture mesh -> Enable Impact Damage -> set Impact Damage = Damage Threshold
5. Drag the new object into the world and hit it with a catapult shot
